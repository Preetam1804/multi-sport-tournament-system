# Sports Management System

Project structure:

- src/ui/  -> Swing UI
- src/db/  -> DB connection
- src/model/ -> Classes (Team, Player, etc.)
- src/dao/ -> Database operations
- src/main/ -> Main app entry
- lib/mysql-connector.jar -> MySQL JDBC driver
- database/schema.sql -> DB schema
- database/sample_data.sql -> Seed data
